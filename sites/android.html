Android Application Development

Overview

Throughout my android development course I was tasked with developing and designing several Java-based Android applications in Android Studio, applying the MVVM architecture to ensure clean code separation and long-term maintainability. My final project was a secure password manager app that utilized a master password which was required to view other passwords and local database storage to protect user credentials. I also designed and implemented a custom scientific calculator featuring BEDMAS parsing logic to handle complex arithmetic operations beyond the capabilities of a standard calculator project. Accross these projects I integrated dynamic and responsive UI components to deliver a smooth and intuitive mobile user experience.
